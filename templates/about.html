About

All of us like creating applications and working on our projects. In a bid to increase open source contributions I thought of providing a platform to share our projects that we work on.

How to add your project

Sharing project is fairly easy, and takes some inspiration from the model implemented by hackercouch.
Instead of filling a form, you just need to fork repo, update a file and submit a Pull Request.
Since many of us have a working knowledge of GitHub, this seemed to me as a viable model.

Proper instructions on adding a project can be found at the GitHub Repo.
